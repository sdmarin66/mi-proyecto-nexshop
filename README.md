# 🛒 Proyecto Base de Datos - NexShop Group S.A.

**Alumno:** Sergio Durán Marín  
**Descripción:** Diseño, estructuración relacional e implementación física de la base de datos de la empresa.

## 🏢 Sobre la Empresa
NexShop Group S.A. es una empresa de distribución híbrida fundada en 2015. Cuenta con una plataforma de venta online y tres tiendas físicas en Valencia, Madrid y Barcelona que comparten catálogo pero funcionan de manera independiente.

## 🚀 Instrucciones de Ejecución
1. Descargue los archivos de este repositorio.
2. Abra su gestor de bases de datos (como PHPMyAdmin) e importe primero el archivo `/sql/schema.sql` para crear las tablas.
3. Importe el archivo `/sql/datos.sql` para cargar los datos de prueba realistas.
4. Vaya a la carpeta `/consultas/` y ejecute el código de `consultas.sql` de forma individual para obtener las respuestas del negocio.
